#

# - Description of the vulnerability
# - Steps to reproduce the issue
# - Potential impact assessment
# - Any proof-of-concept code (if applicable)
#
# What you can expect from us:
# - Acknowledgment within 24 hours
# - Regular updates on our progress
# - Credit in our security acknowledgments (if desired)
# - Responsible disclosure timeline coordination
#

# - Production infrastructure and services
#
# Out of scope:
# - Social engineering attacks
# - Physical security issues
# - Third-party services not controlled by XEUR.AI
# - Issues affecting end-of-life software versions
#
